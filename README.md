Introduction
SmartWrite is an AI SaaS platform developed by Leen-Co Tech Ltd to help students efficiently write research project documentations. The platform supports various project formats and offers AI assistance, collaboration tools, a plagiarism checker, and citation management.

Features
User Authentication: Secure user registration and login functionality.
Project Formats Management: Predefined project formats and the ability to customize formats.
AI Assistance: AI capabilities to assist users in generating content for their project documentation.
Collaboration Tools: Collaboration features for users to work together on research projects.
Plagiarism Checker: Integration of a plagiarism checker API to ensure content originality.
Citations and References: Features to manage citations and references in project documentation.

Usage
Start the development server:
bash
Copy code
python manage.py runserver
Access the application in your web browser at http://127.0.0.1:8000/.

Register a new user or log in using existing credentials.

Explore the platform's features, such as selecting and customizing project formats, using AI assistance, collaborating with others, and managing citations and references.
